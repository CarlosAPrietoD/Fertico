POS Stock

Basic feature

  • Display of current stock of products in running session.
  • Seller can select Stock location.
  • Select stock type for display product stock in POS.
  • Seller can deny or allow order for negative stock.
  • Seller can set the stock limit below which order will be denied.
  • Display of Error message for products lesser in stock.

Seller can display product stock in POS and can also select the stock type to display
